## Dependencies

| Project             | Version |
|---------------------|---------|
| rust-lang.org/cargo | ^0.65   |

## Getting Started

Direct install using cargo:

```sh
cargo build --release
cargo install --path .
```

## Updating dependencies

```sh
cargo update
```
